# HelloAGENTS 知识库入口（GameKnowledge-Base）

本目录用于存放由 HelloAGENTS 维护的结构化知识与交付物（计划书 / 模块文档 / 归档），以便让“代码事实”与“文档事实”长期保持一致。

## 快速入口

- 项目上下文：`helloagents/context.md`
- 工程与发布约定：`helloagents/project.md`
- 变更记录（知识库 SSOT）：`helloagents/CHANGELOG.md`
- 方案包工作区：`helloagents/plan/`
- 模块文档：`helloagents/modules/`
- 归档：`helloagents/archive/`
- Wiki（历史沉淀/碎片知识）：`helloagents/wiki/`
